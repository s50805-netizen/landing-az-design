# AZ Interior Design - Landing Page

A modern, responsive landing page for AZ Interior Design, showcasing interior design services with a focus on small space optimization and premium materials.

## Features

- **Sticky Header** with WhatsApp CTA button
- **Hero Section** with image slider showcasing different interior styles
- **Challenges Section** highlighting customer pain points
- **Selling Points** - Space Saving Design, Premium Materials, Personalized Solutions
- **Case Studies** showcasing completed projects
- **Modules Section** displaying design solutions
- **6-Step Process** explaining the workflow
- **Testimonials** from satisfied customers
- **FAQ Section** answering common questions
- **Lead Generation Form** with WhatsApp integration
- **Responsive Design** for mobile, tablet, and desktop

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (no dependencies)
- WhatsApp Business API integration

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/s50805-netizen/landing-az-design.git
   cd landing-az-design
   ```

2. Add your images to the `assets/images/` directory (see `assets/images/README.md` for required images)

3. Open `index.html` in a web browser or serve with any web server

## Configuration

### WhatsApp Number
The default WhatsApp number is set to `60123456789`. To change it:
1. Update the phone number in `index.html` (all WhatsApp links)
2. Update the phone number in `script.js` (form submission handler)

### Content Customization
- Edit `index.html` to modify content
- Edit `styles.css` to change colors, fonts, and layout
- Edit `script.js` to modify functionality
